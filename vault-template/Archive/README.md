# Archive/

Archived tickets, daily logs, and sessions. Auto-populated by `archive-cleanup.sh`.

```
Archive/
├── tickets/YYYY-MM/    (done tickets, 30+ days old)
├── daily/YYYY/         (daily logs, 30+ days old)
├── sessions/YYYY/      (session archives, 60+ days old)
└── archive-log.md      (auto-generated move log)
```
