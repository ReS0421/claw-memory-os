# Memory/State/

Current domain state files for MEMOS v3.

## Role
- One file per domain
- Replace whole-file state when the domain meaningfully changes
- Keep this focused on **current truth**, not historical detail

## Good fits
- project current status
- infra current status
- knowledge system current status
- investing operating status

## Bad fits
- one-off events
- TODO lists
- verbose meeting notes
- duplicate copies of Topic docs
